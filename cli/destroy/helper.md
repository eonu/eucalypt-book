# Helper

### Command

```ruby
$ eucalypt destroy helper
# Alias/shortened
$ eucalypt d helper
```

### Arguments

* `[NAME]` - The name of the helper \(if argument is empty, a list of existing helpers will be displayed for the user to select one from\).

### Examples

* `$ eucalypt d helper tweet`

  Destroys the `tweet` helper.
