# Model

### Command

```ruby
$ eucalypt destroy model
# Alias/shortened
$ eucalypt d model
$ eucalypt d m
```

### Arguments

* `[NAME]` - The name of the model \(if argument is empty, a list of existing models will be displayed for the user to select one from\).

### Examples

* `$ eucalypt d model tweet`

  Destroys the `tweet` model.
