# Table

### Command

```ruby
$ eucalypt migration drop table
# Alias/shortened
$ eucalypt m drop table
$ eucalypt m drop t
$ eucalypt m d t
```

### Arguments

* `[NAME]` - The name of the table.

### Example

* `$ eucalypt m drop table users`

  Drops the `users` table.
