# Model

### Command

```ruby
$ eucalypt generate model
# Alias/shortened
$ eucalypt g m
```

### Arguments

* `[NAME]` - The name of the model.

### Options/flags

* `--spec`, `--no-spec`, _\(Default: True\)_

  Specify whether or not to generate a spec file along with the model.
